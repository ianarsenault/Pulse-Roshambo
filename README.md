## Run Locally

Clone repo

```sh
$ git clone https://github.com/ianarsenault/Pulse-Roshambo.git
```

To run client

```sh
$ cd client
$ npm install
$ npm run dev
```

[Install MongoDB](https://treehouse.github.io/installation-guides/mac/mongo-mac.html)

To run MongoDB

```sh
$ mongod
```


To run server

```sh
$ cd server
$ npm install
$ npm run dev
```

For PC

```sh
$ npm run dev:pc
```
